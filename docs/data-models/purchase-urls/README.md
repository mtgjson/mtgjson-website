# Purchase Urls

The Purchase Urls Data Model describes url paths to purchase a product from a marketplace.
  
**Parent model:** [Card (Atomic)](/data-models/card-atomic/), [Card (Deck)](/data-models/card-deck/), [Card (Set)](/data-models/card-set/)
