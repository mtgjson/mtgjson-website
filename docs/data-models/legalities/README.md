# Legalities

The Legalities data model describes a list of legalities in  play formats for a specific card.

**Parent model:** [Card (Atomic)](../card-atomic/), [Card (Deck)](../card-deck/), [Card (Set)](../card-set/)  
**Parent property:** `legalities`
